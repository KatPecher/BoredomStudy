Welcome!

This is the positron environment for the boredom study to do the analysis in. This README.md will walk you through the steps and explain what to do. The file "Codebook.md" gives an overview of the codes, variable names and other abbreviations. 


## Starting point
#### Downloading the data and setting things up
All of the data can be found as a handy zip file under this LINK.
Once unpacked into the "data/raw" folder, you can continue to run the scripts "Imports.R" and "Packages.R" to import and load packages.

#### Processing the data
Once the data is in the directory, you can run the script "Processing.R" to - guess what - process the data. This will process the data for each of the three inputs (Limesurvey, Balance and PsychoPy) to remove test runs, faulty data and sort the participant IDs.
